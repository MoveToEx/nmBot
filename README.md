# nmBot

A chatbot for Tencent QQ, with a focus on Long Yutao stickers also bringing other utilities.

## Setup

```sh
$ git clone https://github.com/MoveToEx/nmBot
$ cd nmBot
$ poetry install
$ cp .env.template .env
$ vim .env # Fill in env variables
$ nb run
```
